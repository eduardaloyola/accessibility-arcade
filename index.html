DEVELOPER TRAINING PROTOCOL v1.0

1. Semântica

Leitores de tela navegam por estruturas lógicas. Tags como <h1> situam o usuário cego imediatamente ao carregar uma página.

2. Contraste

Para pessoas com baixa visão ou daltonismo, as cores do texto e do fundo precisam de uma proporção mínima de contraste de 4.5:1 (Regra WCAG).

3. Descrição

Imagens informativas precisam do atributo "alt" preenchido. Tabelas precisam do atributo scope="col" nas tags <th> para associar dados linearmente.
